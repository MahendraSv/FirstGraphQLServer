# FirstGraphQLServer
First GraphQL Server with express, node

## Features
* Express
* Node
* GraphQL
* GraphiQL

## Setup
1. Install [Git]
2. Clone this project: `https://github.com/MahendraSv/FirstGraphQLServer.git`
3. Move to cloned folder
4. Install pacakges `git install`
5. Run the project `npm start`
